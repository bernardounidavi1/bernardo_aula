Gitflow é um jeito organizado de usar o Git para trabalhar em equipe. Ele define onde cada tipo de mudança deve acontecer: tem um lugar para o código que já está no ar, outro para o que está em desenvolvimento e ramificações temporárias para novas funções, testes ou correções urgentes. Seguindo esse fluxo, todo mundo sabe onde mexer e como juntar as mudanças, evitando bagunça e facilitando o trabalho.
